 # in
morg-server
$ npm run build; node dist/bundle.js

in proxy.conf.json, set the target to the server address (where the server is running):

  {
    "/api": {
      "target": "https://amd:8082",
      "secure": false,
      "changeOrigin": true
    },
    "/blogs/api": {    <-- if you have a separate API for blogs morg-server
      "target": "http://amd:3000",
      "secure": false,
      "changeOrigin": true
    }
  }

* Install

- to get ng
$ npm install

  Then run it with npx:

  npx ng serve --host amd --port 4200

  Or use npm scripts in package.json (likely already there):

  npm start
  # or
  npm run start
